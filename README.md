Created following the Hyperskill course "Java Developer". No forms for AI has been used in making this project.

# How to use
To play, write the following line:
start (user/easy/medium/hard) (user/easy/medium/hard)

To exit the program, simply write "exit".

## About

Easy bot - Picks a random space
Medium bot - Is able to predict one move in the future, otherwise defaults to random
Hard bot - Uses the minimax algorithm. Impossible to win against.
